## 💡 Dicas para Melhores Resultados

### 1. Variações
Adicione ao final dos prompts:
- `--seed 12345` (para consistência)
- `--chaos 20` (variações sutis)
- `--stylize 250` (estilo mais artístico)

### 2. Upscale
Após gerar:
- Use `U1`, `U2`, `U3` ou `U4` para upscalar a imagem desejada
- Depois use `Upscale (Subtle)` ou `Upscale (Creative)`

### 3. Ajustes Regionais
Para adicionar variações:
```
[Vary (Region)] - Selecione área para regenerar
[Zoom Out 2x] - Ampliar contexto
[Pan] - Expandir para lado
```

## 🚀 Fluxo de Trabalho Recomendado

1. **Gerar 3 variações** de cada imagem
2. **Selecionar a melhor** (U1-U4)
3. **Upscale** a escolhida
4. **Download** em alta resolução
5. **Otimizar** (compressão para web)
6. **Renomear** com nome descritivo
7. **Upload** para `/public/images/artigos/`

**Tempo estimado:** 2-3 horas para todas as 45 imagens
